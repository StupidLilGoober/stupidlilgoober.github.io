CLI3 is a simple Rust program, developed by me, that plays songs in the terminal. I built it because
I wanted to start 'diversifying' my skillset in programming. Previously, I only really had programs
made in C / C++ (with the exception of my [crappy fast sqrt in rust](https://github.com/StupidLilGoober/fast-sqrt-in-Rust)).

## What I Learned

I learned, with this project, a bit about Rust. I first-hand experienced an issue from the Borrow
Checker, which taught me that only one variable can 'own' a piece of data at a time, so that's
interesting.

I really like Rust. I'll probably make another project in it, but C++ is the ol' reliable.

## Should You Use Rust?

Personally, I say *yes*. It's a pretty fun language, and has actual real-world applications.
Honestly, the project I made wasn't really all that useful, but whatever.

MONTH/DAY/YEAR

A while back, I decided I was going to start making games. Why I did, I don't really know, but game engines (such as Godot and Unity) weren't much fun, so I decided to use SFML.

I don't really like long intros, so I guess I should just get on with what you're really here for.

First, I decided it would be helpful to make a full list of things I would need to program, as that will give me a general idea of how I should program the solutions.

1. Window
2. Grid with integer values representing what's in it
3. Snake that can take up multiple spots
4. Basic controls
5. Fail state

NOTE: I used C++ to program this, but the code examples will use Python to keep things readable.
